# Exmail

腾讯企业邮箱的API接口的调用，详细用法看代码，都有注释的

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'exmail'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exmail
